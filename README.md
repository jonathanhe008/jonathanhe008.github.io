# jonathanhe008.github.io

Personal site for Jonathan He — software engineer, identity and agentic security.
Jekyll + GitHub Pages, live at [hejonathan.com](https://hejonathan.com).

Deep navy + warm gold, with a security-themed micro-interaction system
(padlock unlock, wax-seal break, signature stroke-draw).

## Develop

```bash
bundle install
bundle exec jekyll serve --livereload   # http://localhost:4000
```

Push to `main` and GitHub Pages rebuilds within ~60s. Blog posts live in
`_posts/`; page content in `_layouts/home.html`.

## Inspired by

Structural pattern-language (numbered sections, fixed side rails, tabbed job
history, first-visit loader) adapted from Brittany Chiang's MIT-licensed
[v4 portfolio](https://github.com/bchiang7/v4). No markup was copied; the
palette, typography, and interaction vocabulary are original.
